# Repo2Reel — Generate Animated Architecture Videos from Any GitHub Repo

A one-command tool that turns any GitHub repo into a polished animated architecture walkthrough developers can share on X, LinkedIn, and their README. It rides the trending interest in reusable AI video-production skills and animated architecture diagrams, but differentiates by focusing on repo-to-video automation rather than a generic skill pack.

## Why This Repo

This project is generated by a trend-driven pipeline. It targets **Open-source maintainers, devrel engineers, startup founders, hackathon teams, and technical writers** and focuses on the niche: **AI-assisted developer marketing and documentation videos for open-source projects**.

## Key Features

- CLI takes a GitHub repo URL or local folder and generates a 30–60s animated architecture explainer video as MP4/GIF/HTML
- Static analysis extracts languages, services, dependencies, Docker/CI files, and README concepts into an editable architecture graph
- Hand-drawn animated diagram theme with Mermaid/SVG export, zoom/pan camera moves, captions, and intro/outro templates
- Optional LLM adapter turns repo facts into a concise storyboard and narration script; works offline with heuristic templates if no API key
- Share-ready output: social preview image, README badge, and `docs/architecture-video.md` embed snippet
